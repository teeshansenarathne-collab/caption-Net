# Caption Net - AI Caption Generator

Welcome to **Caption Net**, a professional AI-powered tool for generating video captions.

## How to Run
Simply double-click `index.html` to open the application in your web browser. No installation required!

## Features
- **Fast Upload**: Drag and drop support for video files.
- **AI Simulation**: Visualizes the caption generation process.
- **Design Studio**: Customize your captions with trending templates (Neon, Karaoke, Minimal).
- **Multi-Language Support**: (Simulated) Ready for global audiences.

## Customization
- **Logo**: The logo is built with CSS in `index.html`. You can replace the `<i data-lucide="...">` icons with your own SVG or Image.
- **Background**: The background uses CSS animations. You can change colors in `styles.css`.
- **Styles**: Add more caption templates in `styles.css` under `.caption-style-*`.

## Deployment
To host this online for free:
1. Upload these files (`index.html`, `styles.css`, `script.js`) to **GitHub Pages**, **Netlify**, or **Vercel**.
2. It will work instantly!

Enjoy your new professional web app!
